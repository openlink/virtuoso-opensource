There are two types of news groups: local and remote (mirror).
The local newsgroups can be created and supported from the 'local newsgroups' menu item in UI.
You should add a news server first in order to select a remote news group. To do this click on the 'Add' button below.
After creating the news server, click on server's name to view the list of attached newsgroups.
You can attach newsgroups to the server by clicking on the hyperlink. Limit the newsgroups listing by filling the "match" cases. Press the 'List matching' button to display the filtered newsgroups list.
Attaching of newsgroups to your server is performed by selecting them (checking the selection boxes) and pressing the 'Finish' button.
Check the desired newsgroup and click 'go' button to start immediate update. If you don't do this, you have 30 minutes before automatic update starts. This time can be tuned in newsgroup 'Properties'.
Adding of new newsgroups to the existing list is performed by selecting the 'list' option in listbox and pressing the 'Go' button.
To add a local newsgroups click on the hyperlink in the 'local newsgroups' item in UI. Type a unique name of the newsgroup to be added and click 'Create' button.
